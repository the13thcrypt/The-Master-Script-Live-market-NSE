# The Master Script – Live Market (NSE)

A Python-based script for live market analysis on the National Stock Exchange (NSE) of India.

This project focuses on clean strategy logic, correct position sizing, and capital-aware trading behavior.  
The strategy logic remains the same regardless of account size — only position size scales with capital.

---

## Features

- Live or near-live NSE market data
- Strategy logic independent of capital size
- Correct position sizing
- Risk management friendly structure
- Easy to modify and extend

---

## Files

- `The_Master_Script_Live_market_NSE.ipynb` – Main trading script
